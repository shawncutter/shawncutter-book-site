## Status Workflow

| Status | Meaning | Folder |
|--------|---------|--------|
| idea | Seed concept, rough notes | `idea/` |
| draft | Being written, not ready | `draft/` |
| ready | Complete, ready for Medium | `ready/` |
| published | Live on Medium | `published/` |
| expanded | In a book chapter | `expanded/` |

## How to Review

**To review polished work:** Open any `ready/` folder — these are publication-quality pieces.

**To review drafts that need attention:** Open any `draft/` folder — these range from rough outlines to near-complete pieces.

**To see the plan for an arc:** Open `_meta/{arc}-arc-plan.md`

**To see what questions are still open:** Open `_meta/running-questions.md`
