# **Pads We Can Ship in Six Months** _Standard blocks, honest timelines._

Job trailer heat fights the cold. Laminated templates on the table: 50, 75, 100 MW. Each with a bill of materials, permits, crews, and a calendar measured in days, not hope. I run my finger down the 50: ten 5‑MW gas units with heat recovery, 100 MWh BESS, 34.5‑kV step‑up, greenhouse tie‑in, microgrid controller with a boring UI. Three identical Gantts stacked—Pad A, Pad B, Pad C.

Custom kills schedules. We learned that the hard way. Every special part becomes a special problem. So we standardize what we can stomach. Same switchgear family. Same relays. Same enclosure footprints. Same commissioning scripts. We let local context pick the corridor and the color of the fence. Everything else is repeatable.

The turn is a change order freeze at D‑minus‑45. It saves more than it hurts. If the drawing is wrong, fix the corner next build. Respect the craft. Change the corner first. Logs before lipstick: every acceptance test runs offline, in airplane mode, with a pass/fail that a tired tech can see at 3 a.m.

We set the guardrails: BESS lives between 30–80% except in cold snaps. Black start proven weekly. Recloser curves tested against island transfer. Heat loop holds 170–185°F; greenhouse CO₂ target 800–1,000 ppm. Every pad ships with spares—two controllers, one relay set, three fans, one pump. Four tries on start; then switch plans. Sip‑not‑chug.

Farmer discipline, oilfield tempo, startup bias to ship. EnergiAcres is where they fuse: pads that roll in, stand up, and feed people while they feed power.

**AI tie.** AI builds the repeatable skeleton: it generates the BoM variants, checks vendor lead times, and schedules crews against crane days and cure times. It simulates dispatch for heat/power to validate the loop before steel arrives. It writes the one‑screen SOP, then watches commissioning logs and flags drift. Coach, not king.

**Rule: Standardize everything you can stomach.**

**AI Playbook — Six‑Month Pad**  **Purpose:** Ship a repeatable pad on honest time. **Prompt:**

1. “Given 50/75/100 MW options, generate BoMs with lead‑time sources and a 180‑day schedule (civil → electrical → mechanical → commissioning).”

2. “Create acceptance tests for black start, islanding \<100 ms, SOC policy, heat loop temp/flow, and neighbor circuit continuity.”

3. “Produce a change‑order freeze policy and an exceptions log that defers design tweaks to the next pad.”

4. “Draft a spare‑parts kit and a one‑screen SOP for 3 a.m. techs.” **Constraints:** Change‑order freeze at D‑45 • Airplane‑mode tests • Repeat crews/roles • Spare kit on day 1 • Public ‘pad receipt’ on day 30/60/90/180.
